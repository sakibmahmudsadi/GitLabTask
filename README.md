# 🛒 AIUB MART  
### Campus Mini E-Commerce Platform

AIUB MART is a role-based e-commerce platform designed specifically for the AIUB community. It provides a secure, structured, and user-friendly system for buying and selling academic products such as books, electronics, and study materials.

---

## 🚀 Overview

This platform solves the problem of unorganized trading through social media by offering:
- Centralized product listings
- Secure authentication
- Role-based system management
- Transparent transactions

---

## 👥 User Roles

| Role          | Description |
|--------------|------------|
| 🧑 Customer   | Browse products, add to cart, place orders |
| 🏪 Store Owner | Manage products, stock, and orders |
| 🧑‍💼 Manager   | Handle complaints and monitor activity |
| 🛠️ Admin      | Full system control and user management |

---

## ⚙️ Key Features

- 🔐 Secure Authentication System  
- 🛍️ Product Browsing & Search  
- 🛒 Shopping Cart & Checkout  
- 📦 Order Tracking System  
- 💬 Messaging System  
- ⚠️ Complaint & Dispute Handling  
- 📊 Admin Dashboard
