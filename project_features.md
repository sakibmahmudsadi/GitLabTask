# 🚀 Project Features

This document outlines the main features of **AIUB MART**.

---

## 🔐 Authentication & Security

- User Registration with email verification
- Secure Login system
- Password reset functionality
- Role-based access control

---

## 🛍️ Customer Features

- Browse products by category
- Search products by name
- View product details
- Add/remove items from cart
- Place orders and checkout
- Track order status
- Send messages to store owners
- Submit complaints
- Profile management

---

## 🏪 Store Owner Features

- Add new products
- Edit product details
- Manage stock levels
- Remove products
- View and manage orders
- Reply to customer messages
- Store performance dashboard

---

## 🧑‍💼 Manager Features

- Monitor platform activity
- Handle customer complaints
- Resolve disputes
- Issue warnings to users/stores
- Suspend/unsuspend accounts

---

## 🛠️ Admin Features

- Full system access
- Manage all users and roles
- Approve/reject manager applications
- Monitor all stores
- View system activity logs
- Suspend accounts

---

## 📊 System Features

- Role-based dashboards
- Notification system
- Data validation and error handling
- Secure database management
- Scalable architecture

---

## 🔮 Future Enhancements

- 💳 Online Payment Integration  
- ⭐ Rating & Review System  
- 📱 Mobile App Version  
- 📈 Advanced Analytics Dashboard  

---

## 🎯 Summary

AIUB MART is a **complete role-based e-commerce system** designed for a university environment, focusing on:
- Security
- Transparency
- Usability
- Scalability
